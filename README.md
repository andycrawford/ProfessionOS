# Profession OS

> **Every signal. One surface.**

AI-powered command center for high-output professionals. Connect your email, calendar, code repositories, and CRM — monitored by AI, surfaced in one agentic interface.

---

## What it is

Profession OS is the operating system layer for professional life. Most tools show you data. Profession OS shows you what to do next — by reading across every signal in your work context and surfacing the actions that matter, exactly when they matter.

**Mail · Calendar · Code · CRM — unified, not just aggregated.**

---

## Core capabilities

| Capability | What it does |
|---|---|
| **Activity Timeline** | Real-time feed across all connected sources — one scroll, full context |
| **AI Assistant Panel** | Ask across your entire work graph; get answers grounded in your actual data |
| **Command Center** | Keyboard-driven workspace with composable widgets per role and workflow |
| **Signal Intelligence** | AI surfaces anomalies, blockers, and opportunities before you notice them |
| **Multi-source auth** | Connect GitHub, Google, Linear, Salesforce, and more via OAuth |

---

## Positioning

Profession OS is built for the professional who refuses to operate at half-speed because their tools don't talk to each other. It is operator-grade tooling — not another dashboard, not another feed reader.

**Target audience:** Engineers, managers, founders, operators, and knowledge workers who live inside multiple tools and lose hours to context switching every day.
